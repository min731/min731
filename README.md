## 👋 Introduction

#### 안녕하십니까. AI 기술을 활용하여 더 나은 세상을 만드는 신입 개발자 ‘임정민’입니다. 
#### 최근 1년간 다양한 AI 팀프로젝트를 이끌며 과학기술정보통신부 장관상을 포함한 5개 공모전 수상 실적을 이끌었습니다.
#### 자기계발한 내용들을 모두 깃허브‧블로그에 기록하여 꾸준히 성장할 수 있음을 증명합니다. (개인 블로그 👉 [Link](https://velog.io/@min0731/series) )
#### 5년안에 세계 최고 MLOps 엔지니어가 되기 위해 노력하고 있습니다. (now ~ 2028)
